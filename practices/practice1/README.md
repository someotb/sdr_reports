# Лабораторная работа: Архитектура ADALM PLUTO SDR. GNU RADIO. Построение радио-приёмника

## Описание
Данный проект представляет собой отчёт по лабораторной работе, выполненной в рамках дисциплины SDR (Software-Defined Radio). В работе изучается архитектура SDR-устройства ADALM PLUTO, основы работы с GNU Radio и построение FM-радиоприёмника.

## Цель работы
- Изучить основы SDR-систем и их архитектуру.
- Освоить работу с инструментом GNU Radio.
- Разработать программу для приёма FM-радио с использованием SDR.

## Задачи
1. Изучить теоретические основы SDR.
2. Ознакомиться с архитектурой ADALM PLUTO SDR.
3. Построить схему приёмника в GNU Radio Companion (GRC).
4. Настроить и запустить программу для приёма FM-радио.

## Основные компоненты проекта

### Аппаратная часть
- **ADALM PLUTO SDR** — устройство для приёма и передачи радиосигналов.

### Программная часть
- **GNU Radio** — фреймворк для разработки SDR-приложений.
- **GNU Radio Companion (GRC)** — графический интерфейс для построения схем.

### Блоки, использованные в проекте
1. **PlutoSDR Source** — приём данных с SDR-устройства.
2. **Low Pass Filter** — фильтрация сигнала.
3. **WBFM Receive** — демодуляция FM-сигнала.
4. **Audio Sink** — вывод звука на аудиокарту.
5. **QT GUI Frequency Sink** — визуализация спектра сигнала.
6. **QT GUI Time Sink** — визуализация сигнала во временной области.
7. **QT GUI Range** — ползунок для настройки частоты.

## Результаты
В ходе работы была успешно разработана и запущена программа, позволяющая принимать FM-радиостанции с использованием ADALM PLUTO SDR и GNU Radio. Программа включает визуализацию сигнала и гибкую настройку параметров.

## Заключение
Работа позволила получить практические навыки работы с SDR-оборудованием и GNU Radio, а также углубить понимание принципов цифровой обработки сигналов.
